Tag Name Formatted Tag Description
paragraph <p> The <p> tag defines a paragraph.
div <div> The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
section <section> The <section> tag defines a section in a document.
span <span> The <span> tag is an inline container used to mark up a part of a text, or a part of a document.
